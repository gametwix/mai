\CWHeader{Лабораторная работа \textnumero 4}

\CWProblem{
Необходимо реализовать один из стандартных алгоритмов поиска образцов для указанного алфавита.

{\bfseries Вариант дерева:} Поиск одного образца-маски: в образце может встречаться «джокер» (представляется символом ? — знак вопроса), равный любому другому символу. При реализации следует разбить образец на несколько, не содержащих «джокеров», найти все вхождения при помощи алгоритма Ахо-Корасик и проверить их относительное месторасположение.

{\bfseries Вариант алгоритма:} { \normalfont\ttfamily Последовательность букв английского
алфавита длиной не более 256 символов }

{\bfseries Вариант значения:} { \normalfont\ttfamily Числа в диапазоне от $0$ до $2^{32} - 1$ }
}
